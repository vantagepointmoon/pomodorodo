# Project Documentation

This project was prepared by [Tembo](https://tembo.io).

An entirely Replit vibecoded app for the purposes of testing viability coding entirely on the phone.

The basic functionality:

- to do list
- pomodoro clock
- inspirational quotes and excercises
- toggle between quotes and excercises depending on whether the pomodoro clock is ticking
- basic data persistence (SQLite)
